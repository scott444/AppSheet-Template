Getting Started
Setup Workflow
1
Define your app locally
Create apps/<name>/definition.json using npm run dev -- init -n "MyApp"
2
Validate & generate schema
Run npm run dev -- validate -d apps/MyApp then npm run dev -- gen-schema -d apps/MyApp
3
Create the Google Sheet
Open this Apps Script project and run createSpreadsheetFromSchema(schema) with the contents of your generated sheet-schema.json.
4
Create your AppSheet app
Go to appsheet.com, click Create > App > Start with existing data, and select the Google Sheet you just created.
