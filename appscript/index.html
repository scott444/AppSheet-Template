Getting Started

1. Define your app locally in apps/<name>/definition.json
2. Run npm run dev -- gen-schema -d apps/<name>
3. Call createSpreadsheetFromSchema(schema) with your generated schema
4. Create your AppSheet app pointing at the new Google Sheet
